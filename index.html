The Prototype

PartidoGo

It is a tourism application designed to help users find and explore different tourist destinations in the Partido area, such as beaches, resorts, and other local attractions. The system also allows users to view locations through a map and access important information about each destination.

PartidoGO

PartidoGo is a mobile application designed to streamline tourism in the Partido area of Bicol by allowing users to easily locate and book local resorts and tourist spots. It solves the difficulty of finding reliable travel information in the region, specifically targeting local travelers and tourists looking for a seamless exploration experience. Core features include an interactive map, verified accommodation listings, and a simplified booking system.

Evaluation Methods Used

Heuristic Evaluation

Each group member independently evaluated the prototype using Jakob Nielsen's 10 Usability Heuristics. Issues were rated by severity on a scale of 0–4.
